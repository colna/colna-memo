### 06:00 — 修复:pointerdown 里 await getUserMedia 期间松手导致 setPointerCapture NotFoundError + 卡录制态

- **现象**(用户真机):`ChatVoiceRecorder` `Failed to execute 'setPointerCapture': No active pointer with the given id is found`。
